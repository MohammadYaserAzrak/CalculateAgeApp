# Calculate Age Application
A GUI application that calculates your age in months, weeks, and days using tkinter.

## Prerequisites
- Python 3.x

## How to Run

1. Clone this repository:
```
git clone <repo-link>
```
2. Run the calculate_age_app.py file:
```
python calculate_age_app.py
```
